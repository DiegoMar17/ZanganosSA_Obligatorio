## Bitácora de trabajo

Las sesiones de trabajo y las correcciones por página se encuentran en
[`../Skills/sessions/`](../Skills/sessions/):

  (dashboard, apiarios, colmenas, inspecciones, transhumancia, sanidad,
  calendario, finanzas, inventario, produccion, perfil)
